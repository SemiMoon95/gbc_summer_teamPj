## project 1 : GBC RUN

1. 게임 소개
한 달 동안의 GBC 교육을 토대로 GBC의 정체성을 담은 게임. 전체적인 게임의 스토리는 GBC 교육을 마치는 고스트 신입회원들의 모험기이며, 2D 게임 형식으로 쿠키런의 아성을 뛰어넘을 대작이라고 평가받을 수 있을지도..

2. 사용 tool
유니티 2D 모드 사용

3. 프로젝트의 의의
새로운 프로그램인 unity을 배움과 동시에, 이를 적용할 수 있는 프로젝트라 생각합니다. 또한, 이를 활용하여 동아리 회원들과의 친목형성(ex 점수내기) 등을 추구할 수 있을 것이라 생각합니다. 

## project 2 : Batch file virus 

1. Why: GBC에서도 배우지 않았고 접해볼 기회가 없을 배치파일 명령어를 접해보고 어떻게 사용할 수 있을지 생각할 수 있는 좋은 기회가 될 것 같아서

2. What: 배치파일 바이러스 만들기, 응용

## project 3: [한동 어플 앱 기능 추가] 한동대 건물 소개를 위한 학교 지도

1. 앱 소개
- 모르는 건물을 클릭하면 해당 건물에 대한 상세설명이 있다.
- 모르는 건물이나 강의실이 있을 때 안내하기 아이콘을 클릭하고 강의실을 검색하면 화살표로 안내해준다. 

2. 사용 tool
위치 기반 서비스를 제공해주는 프로그래밍

3. 프로젝트의 의의
앱에 대한 새로운 프로그래밍을 배울 수 있고 새내기들이 학교생활 적응하는데 도움이 될 수 있는 프로젝트라고 생각함.
